Need to have sphinx installed and various packages from https://nbsphinx.readthedocs.io/en/0.3.3/installation.html#nbsphinx-Packages

Need to install theme: conda install conda-forge::sphinx_rtd_theme

Build tips at: https://sphinx-rtd-tutorial.readthedocs.io/en/latest/docstrings.html#the-sphinx-docstring-format

To rebuild documentation go to docs:
rm source/modules.rst
rm source/gbtk.rst
sphinx-apidoc -o ./source ../gbtk
make clean
make html
